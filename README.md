# Business Sales Dashboard – Power BI

## Internship Track
Data Science & Analytics  
Future Interns – Task 1

## Objective
To analyze e-commerce sales data and build an interactive business dashboard
to help decision-makers understand sales performance, profit trends, and
regional/category insights.

## Tools Used
- Power BI Desktop
- Microsoft Excel

## Key KPIs
- Total Sales
- Total Profit
- Total Quantity Sold
- Profit Margin %

## Visualizations
- Sales by Region
- Sales by Category & Sub-Category
- Monthly Sales Trend
- KPI Cards

## Learnings
- Data cleaning and validation
- Creating DAX measures
- Building professional dashboards
- Business storytelling using data
