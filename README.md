##  Project Overview
This project focuses on analyzing e-commerce sales data and building a
professional, interactive **Business Sales Dashboard** using **Power BI**.
The dashboard helps stakeholders understand sales performance, profitability,
regional contribution, and product trends for data-driven decision-making.

---

##  Objectives
- Analyze historical sales data
- Identify best-selling products and high-performing regions
- Understand sales trends over time
- Create business KPIs using DAX
- Present insights through clear visual storytelling
